# Opinio は「採用」をどう捉えているのか

Opinio として採用とは、
入社後に活躍する人材を科学的に見極めるプロセスです。

採用の失敗とは、
「不採用」や「辞退」そのものではありません。

以下の状態こそが、真の失敗だと考えています。

- 評価が面接官の感覚に依存している
- 合否判断の根拠が残っていない
- 同じポジションでも評価基準がバラバラ
- 過去の採用判断を振り返れない

そこで私たちは、
採用判断を「再現可能な構造」にする ATS を考案しました。

採用を、属人的な判断から
データに基づく意思決定へ変えることを目指しています。

## 採用プロセスと意思決定の分離

Opinio ATS では、採用プロセス上の「事実」と、
採用可否という「意思決定」を明確に分離して扱う。

Application.status は、応募者が現在どのプロセスにいるかという
**事実の進行状況**を表すものであり、
採用可否という意思決定は HiringDecision として
**独立して記録**される。

両者を分離することで、

- 判断そのものが適切だったか
- 判断後の運用（連絡・調整・承諾）が適切だったか

を切り分けて検証可能にし、
採用を属人的な感覚ではなく、
再現可能な意思決定プロセスとして扱うことを可能にする。
