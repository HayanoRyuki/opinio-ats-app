現在弊社では、採用業務の効率化を目的とした自社ATS（採用管理システム）「Opinio ATS」を開発しております。

本システムでは、人事担当者の手間を減らすべく、複数チャネルからの応募・候補者データを一元的に取り込む機能を実装しています。その中で、ビズリーチ様のサービスとの連携を最優先で進めたいと考えております。

**ご相談内容：**

1. スカウト反応（「興味あり」「話を聞きたい」等）が発生した際の、Webhook通知またはAPIでのデータ取得の可否
2. 候補者の基本情報（氏名・連絡先等）をAPIで取得する方法の有無
3. API利用に必要な申請手続き・審査期間の目安
4. 技術仕様書やAPIドキュメントの提供可否

**弊社システムの概要：**

- 採用管理システム（ATS）として、応募の一元管理・選考パイプライン管理を提供
- ビズリーチ様からのスカウト反応を自動で取り込み、人事担当者の確認後に選考プロセスへ接続
- データはセキュアな環境（AWS）で管理
